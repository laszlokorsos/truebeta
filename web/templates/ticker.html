About the methodology

All estimates use excess returns — both stock and market returns have the risk-free rate (13-week Treasury bill, ^IRX) subtracted before estimation, consistent with the CAPM framework.

Kalman Beta is estimated using a state-space model where both alpha and beta are state variables following a random walk. The Kalman filter recursively updates estimates at each time step, optimally balancing the model prediction against the new data point. The confidence band reflects the filter's posterior uncertainty about beta. The filter runs on data preceding the display range for warm-up.

Rolling OLS Beta uses a rolling window of ordinary least squares regression on the excess-return market model. The window size can be adjusted using the controls above. It assumes beta is constant within each window — a simplification that can cause lag and discontinuities.

Cumulative Alpha measures the stock's idiosyncratic excess return after stripping out the beta-weighted market component: $(r_i - r_f) - \beta \cdot (r_m - r_f)$, compounded over the display period. Both series start at zero at the beginning of the displayed range for fair comparison.
